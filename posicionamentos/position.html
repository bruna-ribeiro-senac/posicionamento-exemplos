Posicionamentos

A propriedade position especifica o tipo de posicionamento que damos para um elemento:

  • static
  • relative
  • fixed
  • absolute

Os elementos são posicionados usando as propriedades top, bottom, left, right.

Position Static

Por padrão, os elementos já são posicionados de forma estática. Os elementos posicionados de forma estática não são afetados pelas propriedades top, bottom, left, right. Um elemento com position: static; está sempre posicionado de acordo com o fluxo normal da página.

Esta div possui position:static;

Position Relative

Um elemento com position:relative; está posicionado em relação à sua posição normal. As propriedades top, bottom, left e right servem para posicionar o elemento.

Esta div possui um position:relative;

Position Fixed

Um elemento com position:fixed; é posicionado em relação à janela de visualização, ou seja, ele permanece sempre no mesmo lugar. As propriedades top, bottom, left e right servem para posicionar o elemento.

Esta div possui position:fixed;

Position Absolute

Um elemento com position:absolute; é posicionado em relação ao ancestral posicionado mais próximo. Poré., caso o elemento posicionado com absolute não tiver ancestrais posicionados, ele usará o corpo do documento.

Esta div possui position:relative;
Esta div possui position:absolute;
